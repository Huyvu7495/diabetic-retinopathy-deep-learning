Below commands are for Ubuntu 16.04

Libraries for PREDICTION, TRAINING and TESTING
$ sudo apt-get install python3-pip python3-dev 
$ pip3 install numpy
$ pip3 install pandas

Libraries for Plotting barplot
$ pip3 install seaborn
$ pip3 install matplotlib

For more information, please visit official tensorflow webpage:
https://www.tensorflow.org/install/install_linux#InstallingNativePip
